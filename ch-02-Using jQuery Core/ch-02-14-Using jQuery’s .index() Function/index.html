.index() is a method on jQuery objects that's generally used to search for a given element within the jQuery object that it's called on.

NOTE: The .index() method in jQuery returns the position of the first element within the set of matched elements relative to its sibling elements.

This method has four different signatures with different semantics that can be confusing:

  1. .index() with No Arguments
    2.           
            var index = $("#index");
            console.log(index.index()); //1
            console.log(index.first().index()); //1
            var ol = $("ol");
            console.log(ol.index()); //0
            console.log(ol.first().index()); //0
            var noArgs = $("#noArgs");
            console.log(noArgs.index()); //0
            console.log(noArgs.first().index()); //0
  2. .index() with a String Argument
    3.           
            var index = $("#index");
            console.log(index.index("div")); //1
            console.log(index.first().index("div")); //1
            var ol = $("ol");
            console.log(ol.index("ol")); //0
            console.log(ol.first().index("ol")); //0
            var strArgs = $("#strArgs");
            console.log(strArgs.index("#strArgs")); //0
            console.log(strArgs.first().index("#strArgs")); //0

    When .index() is called with a string argument, there are two things to consider:

    • First, jQuery will implicitly call .first() on the original jQuery object. It will find the index of the first element. This is inconsistent, so be careful here.
    • The second point to consider is that jQuery is querying the entire DOM using the passed in string selector and checking the index within that newly queried jQuery object. For example, when using .index( "div" ) in the last example above (var index = $("#index"); console.log(index.index("div"));), jQuery is selecting all of the < div > elements in the document, then searching for the index that contains the first element in the jQuery object .index() is called on.
  3. .index() with a jQuery Object Argument
    4.           
            var ol = $("ol");
            console.log(ol.index(ol)); //0
            console.log(ol.first().index(ol)); //0
            var noArgs = $("#noArgs");
            console.log(noArgs.index(noArgs)); //0
            console.log(noArgs.first().index(noArgs)); //0
  4. .index() with a DOM Element Argument

    5. In this case, the DOM element that's passed into .index() is being checked against all of the elements in the original jQuery object. Once all other cases are understood, this should be the simplest case. It is very similar to the previous case, except since the DOM element is passed directly, it is not taken from a jQuery object container.
